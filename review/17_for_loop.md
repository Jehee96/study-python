# for 반복문
● 똑같은 문장을 10번 출력하려면 복사 붙여넣기를 10번 하면 되겠지만 1000번을 출력하려면 그냥 복붙가지고는 안된다.
이 때 필요한게 반복문이다.

```
01. for 변수 in 반복 범위 또는 대상을 적고
02. 콜론(:)을 한 뒤에 다음 줄에 들여쓰기를 하고 나서 반복 수행할 문장들을 적으면 된다.

for 변수 in 반복 범위 또는 대상:
   반복 수행 문장
```
