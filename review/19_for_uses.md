# for 활용
```
for 변수 in 반복 범위 또는 대상:
   반복 수행 문장

* for 반복문은 이런 형태로 쓸 수 있는데 반복 범위 또는 대상을 순회하면서 for문 내의 문장을 반복수행하게 된다.

* 앞에서 range()를 통해 반복범위를 지정해 봤었는데, 이번에는 반복 대상을 이용해서 for문을 활용할 것이다.
```
##
● 반복 대상에는 자료형을 공부할 때 배웠던 **리스트, 튜플, 딕셔너리가** 쓰일 수 있다.
##
**01. for list**
```
* my_list 라는 리스트가 있을 때 for 반복문은 이렇게 쓸 수 있다.
* 이 때 my_list 안에 있는 1, 2, 3 이라는 값들을 순서대로 하나씩 꺼내 가지고 x에 집어넣게 된다.

my_list = [1, 2, 3]
for x in my_list:
   print(x)

> 1
  2
  3
```
##
**02. for tuple**
```
* 튜플도 리스트와 동일하게 사용할 수 있고 예제의 결과도 my_list와 동일하다.

my_tuple = (1, 2, 3)
for x in my_tuple:
   print(x)

> 1
  2
  3
```
##
**03. for dictionary**
```
* 딕셔너리는 조금 다르다.
  이렇게 person 이라는 딕셔너리가 선언되어 있을 때 딕셔너리는 key와 value가 쌍으로 이루어지기 때문에
  person.values() 를 반복 대상으로 하게 되면 person 이라는 딕셔너리가 가지는 값에 해당하는 부분이 출력이 된다.

ex)
person = {'이름' : '나귀욤', '나이' : 7, '키' : 120, '몸무게' : 23}
for v in person.values():
   print(v)

> 나귀욤
  7
  120
  23
```
```
* values() 대신 keys()를 입력하면 key에 해당하는 부분이 출력된다.

ex_02)
person = {'이름' : '나귀욤', '나이' : 7, '키' : 120, '몸무게' : 23}
for k in person.keys():
   print(k)

> 이름
  나이
  키
  몸무게
```
```
* 또는 딕셔너리가 가지는 items() 를 이용하면 key와 value 가 함께 반환이 되어서 모두 다 출력할 수 있다.

ex_03)
person = {'이름' : '나귀욤', '나이' : 7, '키' : 120, '몸무게' : 23}
for k, v in person.items():
   print(k)

> 이름 나귀욤
  나이 7
  키 120
  몸무게 23

* 이때는 key와 value 를 모두 받기 위해서 for 뒤의 변수 부분에 key를 위한 k, value를 위한 v가 콤마(,)로 구분되어서 둘 다 들어간다는 점을 주의해야 한다.
```
##
**04. for string**
```
* 이런 자료형 외에도 일반 문자열도 반복문의 대상으로 사용할 수 있다.
* fruit 이라는 변수에 'apple'이라는 문자열을 선언해두고 반복문을 실행해보면
  이렇게 문자열 내의 글자들이 하나씩 출력되는 것을 알 수 있다.

fruit = 'apple'
for x in fruit:
   print(x)

> a
  p
  p
  l
  e
```
